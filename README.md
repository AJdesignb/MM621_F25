## About the Project

An interactive application that generates randomized character expressions at the press of a button.

This project explores the creative potential of randomness in visual design, utilizing the `random()` function, arrays, and for loops to create dynamic, ever-changing character states. Each button press triggers a new combination of facial features and expressions, demonstrating how algorithmic randomness can produce unexpected and engaging visual outcomes.

Built as an exercise in combining graphics with interactive functionality, the randomizer showcases how simple user input can unlock endless variations, transforming a static character into a living, evolving creation with each interaction.
